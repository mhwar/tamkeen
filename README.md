# منصة تمكين 

منصة لإدارة رحلة الجمعيات الخيرية السعودية بعد الترخيص.

## المحتوى

- **خطة التأسيس**: 22+ مهمة موزّعة على 14 أسبوعاً
- **التقويم التنظيمي**: المواعيد الإلزامية الشهرية
- **اللوائح والسياسات**: 18 لائحة مع نظام متغيرات عالمية
- **المكتبة التشريعية**: 9+ مراجع تنظيمية
- **الشركاء والمزوّدون**: دليل خدمات
- **التقييم الذاتي**: معايير المركز الوطني الثلاثة
- **الخطة التشغيلية**: للسنة الأولى

## هيكل المشروع

```
tamkeen/
├── index.html          الصفحة الرئيسية
├── assets/
│   ├── style.css       جميع التنسيقات
│   └── app.js          جميع التفاعلات
├── data/
│   ├── vars.json       المتغيرات العالمية (24 متغير)
│   ├── regulations.json  قائمة اللوائح (18 لائحة)
│   └── tasks.json      هيكل المهام
├── docs/               التوثيق
├── netlify.toml        إعدادات النشر
└── README.md
```

## النشر على Netlify

1. ارفع المجلد على GitHub في repository باسم `tamkeen`
2. في Netlify: **Add new site → Import from Git**
3. اختر الـ repository
4. الإعدادات الافتراضية تعمل تلقائياً (publish directory: `.`)
5. اضغط **Deploy site**

سيُعطيك Netlify رابطاً مثل: `https://tamkeen-XXXX.netlify.app`

## التطوير المحلي

افتح `index.html` مباشرة في المتصفح — لا حاجة لخادم.
