# دليل البدء السريع — منصة تمكين

## الخطوة 1: تثبيت Claude Code

افتح هذا الرابط واتبع التعليمات:
https://docs.claude.com/en/docs/claude-code/quickstart

أو ابحث في تطبيق Claude عن "Claude Code" واختر إصدار VS Code (الأسهل لمن لا يستخدم التيرمينال).

## الخطوة 2: إنشاء حساب GitHub

1. افتح: https://github.com
2. اضغط "Sign up"
3. أنشئ حساباً مجانياً

## الخطوة 3: ربط Netlify بـ GitHub

1. ادخل على حسابك في Netlify: https://app.netlify.com
2. اضغط: **Add new site → Import an existing project**
3. اختر: **Deploy with GitHub**
4. وافق على ربط الحسابين

## الخطوة 4: نقل ملفات منصة تمكين

1. حمّل ملف `tamkeen-package.zip` من Claude
2. فُكّ الضغط — ستحصل على مجلد `tamkeen`
3. افتح Claude Code (إن استخدمت VS Code: File → Open Folder → اختر مجلد tamkeen)
4. اكتب لـ Claude في المحادثة:

   > "ارفع هذا المجلد على GitHub في repository جديد باسم tamkeen ثم اربطه بـ Netlify"

Claude Code سيتولى كل شيء: إنشاء الـ repository، رفع الملفات، إعداد Netlify.

## الخطوة 5: الحصول على رابطك العام

بعد دقائق سيعطيك Netlify رابطاً مثل:
```
https://tamkeen-أنموذج.netlify.app
```

شارك هذا الرابط مع فريقك — يفتح من أي جهاز.

## كيف تُعدِّل لاحقاً؟

اكتب لـ Claude Code أي تعديل تريده:

- "أضف مهمة جديدة في خطة التأسيس بعنوان كذا"
- "غيّر اسم الجمعية في كل مكان لـ جمعية البر بعسير"
- "أضف لائحة جديدة باسم سياسة المتطوعين"

كل تعديل يُحفظ تلقائياً على GitHub، وNetlify ينشره في 30 ثانية.

## أسئلة شائعة

**س: هل أحتاج دفع شيء؟**
ج: لا — Netlify و GitHub مجانيان لهذا النوع من المواقع.

**س: هل يفتح من الجوال؟**
ج: نعم، الموقع متجاوب مع كل الشاشات.

**س: ماذا لو أردت إيقافه؟**
ج: من Netlify: Site settings → Delete site. ينتهي كل شيء.

**س: من يرى الموقع؟**
ج: أي شخص لديه الرابط. يمكن إضافة كلمة مرور لاحقاً عبر Netlify.
